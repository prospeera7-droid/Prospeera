# **App Name**: Prospeera: Freelancer Financial Services

## Core Features:

- Homepage Hero: Displays a rotating black sphere animation and service CTAs.
- Service Display: Lists accounting/financial services with clear descriptions.
- Contact Form: Collects user inquiries and contact information.
- Case Study Showcase: Highlights success stories and links to detailed views.

## Style Guidelines:

- Background: White (#FFFFFF) for a clean, open feel.
- Primary text: Black (#000000) for maximum readability and contrast.
- Font: 'Source Code Pro' (monospace) for a clean, modern look. Note: currently only Google Fonts are supported.
- Use transparent cards to display content blocks and sections for a minimalist design.
- Implement subtle hover effects on buttons and cards to provide visual feedback and engagement.
- The homepage includes a rotating black sphere for visual engagement.
